# WURDKA Official Website

A modern, high-performance, and interactive web platform built for the **WURDKA** company. This project features a cutting-edge Cyber-AI aesthetic with advanced UI/UX animations and hardware-accelerated rendering.

## 🚀 Tech Stack
* **Frontend:** React.js, Vite
* **Styling:** Tailwind CSS
* **Interactive UI:** `react-tsparticles` (Neural Network Background), Custom Spotlight Cursor, 3D Tilt Effects
* **Form Handling:** Web3Forms API
* **Deployment:** Vercel

## ✨ Key Features
* **Immersive UI/UX:** Features interactive neural network particles, a custom spotlight cursor, and 3D card tilt effects for a premium user experience.
* **Performance Optimized:** Implements native image compression, hardware acceleration (`transform-gpu`), and crisp edge rendering to ensure high-quality visual assets with minimal load times.
* **Security-First:** Secure routing for external documents (preventing Tabnabbing via `rel="noopener noreferrer"`) and secure API key integration.
* **Responsive Architecture:** Fully adaptable layout for Web, Tablet, and Mobile environments.
